5 Day Split

This five day split program is designed for an experienced gym user focusing on hypertrophy. By isolating a specific muscle group in each session, and using approaches such as pyramids, drop-sets and supersets, this program aims to completely exhaust each muscle group, to stimulate growth.

  • Mon: Legs
  • Tue: Chest
  • Wed: Back
  • Fri: Shoulders
  • Sat: Arms
